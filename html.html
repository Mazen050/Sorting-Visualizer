Bubble Sort

Time Complexity: O(n²)

Space Complexity: O(1)

Best Use Case: Educational purposes and small datasets where simplicity is preferred (Arranging Books by Height)

Selection Sort

Time Complexity: O(n²)

Space Complexity: O(1)

Best Use Case: Environments like flash memory where reducing the number of write operations is more important than speed (Simple comparison & Memory writes)

Insertion Sort

Time Complexity: O(n²) | Best: O(n)

Space Complexity: O(1)

Best Use Case: Sorting a hand of playing cards according to the numbers and colors

Merge Sort

Time Complexity: O(n log n)

Space Complexity: O(n)

Best Use Case: Sorting linked lists or large datasets with guaranteed performance (Sorting large lists)

Quick Sort

Time Complexity: O(n log n) | Worst: O(n²)

Space Complexity: O(log n)

Best Use Case: In computer graphics, QuickSort is used for image rendering

Heap Sort

Time Complexity: O(n log n)

Space Complexity: O(1)

Best Use Case: Situations requiring guaranteed O(n log n) performance without recursion like (Traffic management systems)
